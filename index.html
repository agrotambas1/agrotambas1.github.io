Latar Belakang

HTML adalah kepanjangan dari Hypertext Markup Language dan merupakan salah satu bahasa yang paling banyak digunakan dalam mebuat halaman web. Hypertext mengacu pada cara di mana halaman web (dokumen HTML) dihubungkan. Jadi, link yang tersedia pada halaman web disebut Hypertext. Seperti namanya, HTML adalah bahasa Markup yang berarti kamu menggunakan HTML hanya untuk “mark-up” dokumen teks dengan tag yang akan memberitahukan browser struktur untuk menampilkan sebuah desain layout web.
